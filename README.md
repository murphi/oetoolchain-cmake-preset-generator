# `oetoolchain-cmake-preset-generator`

Generate CMake preset from OpenEmbedded / Yocto toolchain

Copyright (C) 2025 Martin Engelmann <murphi@posteo.de>

## Usage

`oetoolchain-cmake-preset-generator` can be used to generate a CMake preset file based on the environment setup script of a SDK generated by Yocto / OpenEmbedded. The generated configure preset is intended to be inherited from in a project specific preset. It defines the same environment variables as the environment setup script.

If `-` is passed for `--output` the result is printed to stdout.

```
usage: oetoolchain-cmake-preset-generator [-h] [--version] -i INPUT [-o OUTPUT] [-n NAME] [-g GENERATOR]

Generate CMake preset from OpenEmbedded / Yocto toolchain

options:
  -h, --help            show this help message and exit
  --version             show program's version number and exit
  -i, --input INPUT     path to environment setup script
  -o, --output OUTPUT   Path to generated CMake preset
  -n, --name NAME       Name of the generated preset (default=oe-toolchain)
  -g, --generator GENERATOR
                        Generator for the preset (default=Ninja)
```
